“Go to the Google APIs Console. 
1. Create a new project. 
2. Click Enable API. Search for and enable the Google Drive API. 
3. Create credentials for a Web Server to access Application Data. 
4. Name the service account and grant it a Project Role of Editor. 
5. Download the JSON file. 
6. Copy the JSON file to your code directory and rename it to client_secret.json 

Find the  client_email inside client_secret.json. Back in your spreadsheet, click the Share button in the top right, and paste the client email into the People field to give it edit rights. Hit Send. 
If you skip this step, you’ll get a gspread.exceptions.SpreadsheetNotFound error when you try to access the spreadsheet from Python”
